A) How to Open Project? - top

1.Open the Unity Hub and Click Add

2.Select The Folder that Contains Assets Folder

3.Select the Apporopiate Target Platform and Unity Version(Unity will automatically set unity version if you already installed same version)

B) How to Edit Project? - top

1.Go to File->Build Settings and click Player Settings

2.Change App Name,Company name,Icon,Package Name

C) How to Export Apk? - top

1.Go to File->Build Settings and click Player Settings

2.Set the Keystore and password Then Click Build

D) How to Change Admob? - top

1.Set the App ids in Mobile Ads Setting Assets->Google Mobile Ads->Setting

2.Set the ads ids in Settings MyGames->Settings

So, Replace with your own ids the current ids are test ids of admob you can use to test it. Check Video Here

E) How to Change Unity Service - top

1. Open the Windows->General->Services

2.Create the new Project in service (Remember You have to login with unity account if not)

3.Activate the Ads and InApp Purchase

Set your service before test. Check Video Here

Make sure you don't have any Errors in console. if you got errors disable and enable the ads in service with restart unity until error goes away.

F) Change Other Settings - top

1. Open the MyGames->Settings

2. Set the inapp purchase id - you have to setup the developer console(Apple/GooglePlay) to get in app purchase ids. if you don't want InApp Purchase you can jus disable it in Settings

3. Set the Private Policy Url - if you don't need you can disable it

4.Setup Ios App Id - If you are plan to publish ios set the app id. No need for android

5.Change the Ads Frequency - (Optional)

6.Change Rewards Amount - You can set the reward diamond Daily reward will randomly selected
